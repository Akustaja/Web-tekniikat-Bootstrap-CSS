This is a bootstrap & CSS demonstration

The layout is built using a bootstrap grid. This page contains 3 rows and a total of 4 columns, with the middle row containing two columns.

The style code is written using SASS. A mixin was used to simplify basic setup of the grid's rows with parameters that determine its height, width, background color as well as text color.

These 'p' -elements are styled through the use of a '~' -selector and the 'h2' -element was selected with a '>' -selector.

This column's style is determined through a '+' -selector.

This text changes color when you hover over it, through the use of a ':hover' pseudo-selector. Its size and position were changed by using an attribute selector.
